Python Spider 获取网页内容

发表于 2018-08-01

Intellij IDEA下搭建scrapy项目，并且简单获取网页内容。

  1. 创建scrapy项目，项目名称：SZT_Spider。

    scrapy startproject SZT_Spider

  2. 进入项目。

    cd SZT_Spider

  3. 创建一个爬虫。

    scrapy genspider TicketInfoSpider shenzhentong.com

    爬虫名称：TicketInfoSpider

    爬取的网站域名：shenzhentong.com

Python Spider 环境搭建

发表于 2018-07-31

Python 爬虫框架（Scrapy）安装。

  1. 安装pip

    sudo easy_install pip

  1. 安装Scrapy

    pip install Scrapy

    如果遇到安装six失败，可以：

    sudo pip install Scrapy --upgrade --ignore-installed six
